Bluestone Script

Commands

Here you will find a list off added or extended commands that you can use.

Everything between [] is required and everything between < > is optional.

as

as [selectors] run ...

Will execute the command that comes after for every selector that is found.


as @a[tag=fighter] || @a[tag=barbarian] run say Cracking bones

call

call [function]([parameters])

Executes a local function defined using the 'local' keyword.


call greet(Steve)

data

... set from entity score [data type] [scale] [selector] [scoreboard]

Set's a NBT value to the value of a scoreboard


data modify block ~ ~-3 ~ Items[0].Count set from entity score int 1 @p itemCount

for

for [entry] in [array] {[commands]}

Loops over the array and compiles the commands with that entry


for opPlayer in [Steve, Alex, Notch] {
    effect give $opPlayer protection 100000 100 true
}

holding

holding [item] <nbt> run [command]

Checks wether the selector is holding the specified item.


as @a run holding minecraft:clock run say tick tock

if

if [scoreboard | number] [operation] [scoreboard | number] run [command]

Checks wether the score or value of the current entity is greater than the other score or value


as @a run if timer >= 20 run say timer finished
as @a run if timer >= timerLimit run say timer finished

local

local [name] = [value | function]

Defines a local which can be called.


local playername = Steve
say $playername

Defines a function that can be called using the 'call' keyword.


local greet = (playername) {
    say hey, $playername
    say hi, $playername
    say hello, $playername
}

log

log [scoreboard] <selector> <prefix>

Gets the current value of the scoreboard and shows it in chat


log timer @s Left
