Bluestone script

Get it done

Using BSS you can stop having to worry about tabs and creating seperate files for different statements.

Bluestone Script

{{ if i > 10 => }} <<
 say hey
 say hi
 say hello
>>

Output

execute as @s[scores={i=11..}] run function example:bss_generated/1110

Write less

BSS makes sure you can focus on what you are making by making you write less code.

Bluestone Script

{{ var score = 10 }}
{{ expression score * 2 }}

Output

# Adds the required scoreboards
scoreboard objectives add score dummy
scoreboard objectives add n2 dummy

# Sets the values
scoreboard players set n2 2
scoreboard players set score 10

# And then does the operation
scoreboard players operation @s score *= @s n2
