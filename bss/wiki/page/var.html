Basic usage

Using 'var' is like using a more advanced version of 'scoreboard'. It automatically defines scoreboards for you and can even add them to a seperate file.

If you look at the example below. It creates the scoreboard and also assignes the value to 10.

Bluestone Script

{{ var test = 10 }}

Output

scoreboard objectives add test dummy
scoreboard players set @s test 10

But we can do more. We can actually do some basic operations.

Bluestone Script

{{ var test = 10 }}
{{ var test += 20 }}

Output

scoreboard objectives add test dummy
scoreboard players set @s test 10
scoreboard players add @s test 20
# Expected output sould be 30

Defining types

You might have noticed that it always defines the scoreboard using the dummy type. You can change the type by first defining the scoreboard and then assigning a value to it.

Bluestone Script

{{ var deaths as deathCount }}
{{ var deaths = 10 }}

Output

scoreboard objectives add deaths deathCount
scoreboard players set @s deaths 10
