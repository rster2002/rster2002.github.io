Basic usage

'If' is used to check if a entity has a specific score. In the example below we check if '@a' has a score bigger than 10.

Bluestone Script

{{ as @a => }} <<
 {{ if i > 10 => }} <<
  say hey hi hello
  {{ var i = 0 }}
 >>
>>

Or / And

You can use '||' or '&&' to specify that it should check for both or it should check if any of the statements are true. You can't use both '&&' and '||' in one statement.

Bluestone Script

{{ as @a => }} <<

 # Checks if 'i' is bigger than 10 or if 'r' is smaller than 4
 {{ if i > 10 || r < 4 => }}
  say hey hi hello
  {{ var i = 0 }}
 >>
>>

Bluestone Script

{{ as @a => }} <<

 # Checks if 'i' is bigger than 10 and 'r' is smaller than 4
 {{ if i > 10 && r < 4 => }}
  say hey hi hello
  {{ var i = 0 }}
 >>
>>

Bluestone Script

{{ as @a => }} <<

 # This will give a compiling error
 {{ if i > 10 || r < 4 && c >= 90 => }}
  say hey hi hello
  {{ var i = 0 }}
 >>
>>
